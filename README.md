## Description:

It's an API for a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list. This project allowed me to familiarize myself more with MongoDB and how to understand the backend structure for a social media platform.

## Technology:

Project is created with:

- Javascript
- Node.js
- Express.js
- MongoDB
- Mongoose

## Installation

To run this project, install it locally using npm:

```
npm install
```

## Usage

After installing npm packages, the application will be invoked by using the following command:

```
npm start
```
